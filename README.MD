
RecyclerView

This project implements an endless list of beacons using RecyclerView. A beacon is a Bluetooth Low-Energy device which constantly emits Bluetooth signals. It’s commonly used to identify an object or a location. Each beacon contains 3 values in its identity, namely UUID, Major and Minor. 
•	UUID (Universally unique identifier) is 36 characters long
•	Major and Minor are positive integer values of between 0 and 65535 (both inclusive)

Objectives:
•	Learn how to render a simple RecyclerView with a custom layout.
•	Learn how to implement an adapter class and add list divider.
•	Learn how to implement endless scroll list. 
•	Learn how to implement OnClickListener for list item and UI components in list item.
